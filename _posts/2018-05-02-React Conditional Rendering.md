##### 条件渲染  
<p2> 
**条件渲染**(Conditional Rendering)就是根据条件有选择性地渲染对应的模块儿
</p2>

1. 常用`if`和`condition ? true : false`来控制；
2. `{condition && component}`只用判断condition的真假即可；
3. `{condition ? true : false}`可直接用于HTML的标签内容；
4. 防止渲染可以返回`null`来代替其他渲染输出。
